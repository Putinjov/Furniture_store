# Furniture Store Frontend (Expo)

Мобільний frontend для Furniture Store на Expo/React Native.

## Запуск локально

1. Встановити залежності:
   ```bash
   npm install
   ```
2. Створити `frontend/.env` (або передати env у shell) і вказати:
   ```env
   EXPO_PUBLIC_BACKEND_URL=http://localhost:8000
   ```
3. Запустити:
   ```bash
   npm run start
   ```

## Корисні скрипти

- `npm run start` — запуск Expo
- `npm run android` — запуск для Android
- `npm run ios` — запуск для iOS
- `npm run web` — запуск веб-версії
- `npm run lint` — lint
